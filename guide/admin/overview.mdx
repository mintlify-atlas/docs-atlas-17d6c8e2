The admin dashboard gives you complete visibility and control over D&D Textil. From a single view you can track platform-wide metrics, navigate to any management module, and monitor recent activity.

<Info>
The admin panel is only accessible to accounts with the `admin` role. Sellers cannot access `/admin` routes.
</Info>

## Route

`/admin` — requires role `admin`.

## Global metrics

Four metric cards appear at the top of the overview:

| Metric | Description |
|---|---|
| **Ventas Totales** | Platform-wide revenue with a trend indicator (+12.5%) |
| **Total Usuarios** | All registered users, with active seller count as subtitle |
| **Pedidos** | Total order count with average ticket value |
| **Reportes Calidad** | Total quality reports; subtitle shows open vs. in-review/resolved split |

A **7-day sales line chart** follows the metric cards, then a two-column layout shows the **Top 5 Sellers** ranking and the **Recent Activity** feed (last 6 platform events).

## Carousel management (`/admin/carrusel`)

The carousel editor lets you maintain the homepage promotional slides. Each slide has a **title**, **subtitle**, and **image**. You can add a new slide with the **Agregar Slide** button, edit an existing slide inline (title, subtitle, and image upload), or delete a slide after confirming the prompt.

## Home page visibility (`/admin/home`)

Toggle four homepage sections on or off without a deployment:

| Section ID | Name | Default |
|---|---|---|
| `hero` | Hero Section | Visible |
| `carousel` | Carrusel de Promociones | Visible |
| `featured` | Productos Destacados | Visible |
| `benefits` | Sección de Beneficios | Visible |

Changes are persisted to the API at `POST /api/config` with key `home_sections_config`.

## System configuration (`/admin/configuracion`)

The system config page exposes global platform settings:

| Setting | Description |
|---|---|
| **Nombre del Sitio** | Display name for the storefront |
| **Color Primario / Secundario / Acento** | Hex color pickers for the brand palette |
| **Tasa de Impuesto (%)** | Applied to all sales at checkout |
| **Costo de Envío ($)** | Flat shipping fee in COP |
| **Umbral Envío Gratis ($)** | Order total above which shipping is waived |
| **Umbral Stock Bajo (metros)** | Meters threshold that triggers low-stock badges |
| **Banner Global** | Enable/disable a site-wide announcement bar with message and type (info, success, warning) |
| **Modo Oscuro por Defecto** | Whether new user accounts start in dark mode |

Click **Guardar Configuración** to persist all changes.
