## Filters and sorting

The `ProductGrid` component (`src/components/ProductGrid.jsx`) manages all filter and sort state locally with `useState` and derives the visible product list with `useMemo`.

### Category filter

Category buttons are generated dynamically from the unique `category` values present in the loaded product list, prefixed with an **All** option (`Todos`). Clicking a category pill sets `selectedCategory` and re-filters the list.

```jsx
const [selectedCategory, setSelectedCategory] = useState('Todos');

// Derived from loaded products at runtime
const categories = useMemo(() => {
  const cats = ['Todos', ...new Set(products.map((p) => p.category))];
  return cats;
}, [products]);
```

### Price range filter

A range slider lets you set a maximum price. The slider runs from `0` to `100,000` COP in steps of `5,000`. The minimum is always `0`; only the upper bound is adjustable.

```jsx
const [priceRange, setPriceRange] = useState([0, 100000]);

// Applied during filtering
filtered = filtered.filter(
  (p) => p.price >= priceRange[0] && p.price <= priceRange[1]
);
```

## Product grid layout

`ProductGrid` renders a responsive CSS grid using Tailwind:

- **1 column** on mobile
- **2 columns** on `sm` (â‰¥640 px)
- **3 columns** on `lg` (â‰¥1024 px)
- **4 columns** on `xl` (â‰¥1280 px)

Each cell is a `ProductCard` component wrapped in a `motion.div` that fades and slides up with a staggered delay (`index * 0.05s`).

## ProductCard component

`ProductCard` (`src/components/ProductCard.jsx`) renders a single product tile with:

- **Thumbnail image** â€” uses `product.images[0]`, falls back to `/placeholder.png` on error
- **Category badge** â€” top-left overlay on the image
- **Low stock badge** â€” top-right red pill labeled *Â¡Ãšltimas unidades!* when `stock < 10`
- **Hover overlay** â€” reveals quick-view and wishlist icon buttons
- **Specs row** â€” shows `width` and `stock` in small text
- **Price** â€” formatted in COP with a *por metro* label
- **Add to cart button** â€” calls `addToCart(product, 1)` and turns green when the item is already in the cart (`isInCart` check)

Clicking the card (outside the add-to-cart button) navigates to `/producto/:id`.
