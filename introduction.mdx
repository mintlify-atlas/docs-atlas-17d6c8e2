# D&D Textil — Tienda Digital

D&D Textil is a complete e-commerce platform built for textile and fabric retail. It supports three distinct roles — **client**, **seller**, and **admin** — each with their own dashboard and feature set, all within a single React application backed by a Java Spring Boot API and a PostgreSQL database.

## Technology stack

| Layer | Technology |
|-------|-----------|
| Frontend framework | React 18 + Vite |
| Styling | Tailwind CSS 3 |
| Animations | Framer Motion |
| Routing | React Router v6 |
| Forms | React Hook Form + Zod |
| Backend | Java Spring Boot 3 |
| Authentication | JWT (JSON Web Tokens) |
| Database | PostgreSQL |
| Password hashing | SHA-256 |

## User roles

<Info>
Every route in the application is protected by a role-aware `ProtectedRoute` component. Users are redirected to the login page if they try to access a route outside their role.
</Info>

| Role | Access |
|------|--------|
| `client` | Shopping, cart, checkout, client dashboard |
| `seller` | Seller dashboard, product management, order fulfillment |
| `admin` | Full access including user management, analytics, and system config |

## Project origin

D&D Textil was developed as a capstone project for the **ADSO program at SENA** (Colombia's national vocational training service), demonstrating a production-grade full-stack application using modern web technologies and software architecture principles.
