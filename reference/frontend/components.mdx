# Component reference

All shared components are exported from `src/components/index.js` and organized into logical groups.

## Layout components

| Component | Route usage | Description |
|-----------|-------------|-------------|
| `Header` | All pages | Top navigation bar with cart count badge, dark mode toggle, and user menu |
| `Footer` | All pages | Site footer with links and branding |
| `ScrollToTop` | App-level | Scrolls to top on route change |
| `AppProviders` | `main.jsx` | Wraps the entire app with all context providers |
| `ProtectedRoute` | `App.jsx` | Guards routes by auth status and user role |

### AppProviders nesting order

```jsx src/components/AppProviders.jsx
export function AppProviders({ children }) {
  return (
    <NotificationProvider>
      <AuthProvider>
        <MetricsProvider>
          <CartProvider>
            <ProductProvider>
              <UIProvider>
                {children}
              </UIProvider>
            </ProductProvider>
          </CartProvider>
        </MetricsProvider>
      </AuthProvider>
    </NotificationProvider>
  );
}
```

<Info>
Providers are ordered so that `AuthProvider` is available to `MetricsProvider` and `CartProvider`, which depend on knowing the current user.
</Info>

## Custom hooks

### `useDarkMode`

Toggles dark mode by adding/removing the `dark` class on `<html>`. Persists the preference in localStorage under the key `darkMode`.

```javascript src/hooks/useDarkMode.js
export function useDarkMode() {
  const [darkMode, setDarkMode] = useLocalStorage('darkMode', false);

  useEffect(() => {
    const root = window.document.documentElement;
    if (darkMode) {
      root.classList.add('dark');
    } else {
      root.classList.remove('dark');
    }
  }, [darkMode]);

  const toggleDarkMode = () => setDarkMode(!darkMode);
  return [darkMode, toggleDarkMode];
}
```

### `useLocalStorage`

A drop-in replacement for `useState` that syncs state with `localStorage`.

```javascript src/hooks/useLocalStorage.js
export function useLocalStorage(key, initialValue) {
  const [storedValue, setStoredValue] = useState(() => {
    try {
      const item = window.localStorage.getItem(key);
      return item ? JSON.parse(item) : initialValue;
    } catch (error) {
      return initialValue;
    }
  });

  const setValue = (value) => {
    const valueToStore = value instanceof Function ? value(storedValue) : value;
    setStoredValue(valueToStore);
    window.localStorage.setItem(key, JSON.stringify(valueToStore));
  };

  return [storedValue, setValue];
}
```

## Utility functions

All utilities live in `src/utils/formatters.js`.

| Function | Signature | Description |
|----------|-----------|-------------|
| `formatCurrency` | `(amount: number) => string` | Formats a number as Colombian Pesos (COP) using `Intl.NumberFormat` |
| `formatDate` | `(date: Date\|string) => string` | Formats a date in Spanish (es-CO) locale, e.g. "16 de marzo de 2026" |
| `generateId` | `() => string` | Generates a unique ID: `{Date.now()}-{random base-36 string}` |
| `truncateText` | `(text: string, maxLength?: number) => string` | Truncates to `maxLength` (default 100) and appends `...` |

## Context providers

| Context | Hook | Key state/actions |
|---------|------|-------------------|
| `AuthContext` | `useAuth()` | `user`, `login()`, `register()`, `logout()`, `hasRole()`, `isAuthenticated` |
| `CartContext` | `useCart()` | `cart`, `addToCart()`, `removeFromCart()`, `updateQuantity()`, `clearCart()`, `totalItems`, `totalPrice` |
| `ProductContext` | `useProducts()` | `products`, `loading`, `error`, `getProductById()`, `refetch()` |
| `MetricsContext` | `useMetrics()` | Platform metrics for admin/seller dashboards |
| `NotificationContext` | `useNotification()` | `showNotification()` with type (success/error/info/warning) |
| `UIContext` | `useUI()` | `isMobileMenuOpen`, `toggleMobileMenu()`, loading state |
