# Frontend stack

The D&D Textil frontend is a single-page application built with React 18, bundled by Vite, and styled entirely with Tailwind CSS.

## npm scripts

```bash
npm run dev      # Start dev server on port 3001
npm run build    # Production build
npm run preview  # Preview production build locally
```

<Note>
The Vite dev server runs on port **3001** by default (`strictPort: false`), so it will find the next available port if 3001 is busy. A proxy is configured to forward `/uploads` requests to the backend at `http://localhost:8081`.
</Note>

## Vite configuration

```javascript vite.config.js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import { fileURLToPath, URL } from 'node:url'

export default defineConfig({
  plugins: [react()],
  resolve: {
    alias: {
      '@': fileURLToPath(new URL('./src', import.meta.url)),
    },
  },
  server: {
    host: '0.0.0.0',
    port: 3001,
    strictPort: false,
    open: true,
    allowedHosts: true,
    proxy: {
      '/uploads': {
        target: 'http://localhost:8081',
        changeOrigin: true
      }
    }
  },
})
```

## Tailwind CSS setup

Tailwind is configured with the `class` dark mode strategy, custom brand colors, and custom font families.

```javascript tailwind.config.js
export default {
  content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}'],
  darkMode: 'class',
  theme: {
    extend: {
      colors: {
        primary: { /* 50–900 orange scale */ },
        secondary: { /* 50–900 zinc scale */ },
        accent: { /* matches primary */ },
      },
      fontFamily: {
        sans: ['Inter', 'system-ui', 'sans-serif'],
        display: ['Outfit', 'Inter', 'sans-serif'],
      },
    },
  },
}
```

## Path alias

The `@` alias maps to `./src`. Use it in any component:

```javascript
import { formatCurrency } from '@/utils/formatters';
import { useAuth } from '@/context/AuthContext';
```
